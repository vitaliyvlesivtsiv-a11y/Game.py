# Прокляття Смарагдових Джунглів

Текстова пригодницька гра, написана мовою **Python**, у якій гравець потрапляє до загадкових джунглів, проходить випробування, збирає предмети та бореться з фінальним босом.

---

## Опис гри

Одного вечора звичайний школяр знаходить старовинний дерев'яний амулет. Варто лише доторкнутися до нього — і герой переноситься до загадкових джунглів.

На початку гри гравець має:

- 100 HP
- порожній інвентар

Під час проходження потрібно:

- досліджувати локації;
- робити вибір у різних ситуаціях;
- збирати корисні предмети;
- знаходити частини Артефакту чотирьох стихій;
- перемогти Кам'яного Стража.

---

## Мета гри

Перемогти **Кам'яного Стража** та отримати **Артефакт чотирьох стихій**.

---

## Запуск

1. Клонуйте репозиторій:

```bash
git clone https://github.com/username/project.git
```

2. Перейдіть до папки проєкту:

```bash
cd project
```

3. Запустіть головний файл:

```bash
python main.py
```

або відкрийте проєкт у VS Code та натисніть **F5**.
